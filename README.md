## Daily Coding Challenges: Logic & Algorithm Lab

### About
This repository is a dedicated space for daily problem-solving. The goal is to solve at least one logic challenge every day using different programming languages (Python, C++, JavaScript, etc.). This practice focuses on optimizing time complexity, mastering data structures, and refining algorithmic thinking.

## This is a continuous exercise in problem-solving and language versatility.

### Repository Structure
The repo is organized by Difficulty or Topic, allowing for a multilingual approach to the same problem:
* */Easy* : Focus on basic syntax and built-in methods.
* */Medium* : Focus on efficient algorithms (Sorting, Searching, Recursion).
* */Hard* : Complex data Structures (Tress, Graphs, Dynamic Programming).
* */Resources* : Cheat sheets and Notes on Big 0 notation and logic patterns.

### Features
* **Multi-Language Solutions**: Many challenges are solved in more than one language to compare performance and syntax.
* **Complexity Analysis**: Each solution includes a brief comment on Time Complexity (O(n)) and Space Complexity.
* **Clean Code**: Prioritizing readability and professional naming conventions.
* **Automated Testing**: (Optional) Basic test cases included for logic verification.

### Languages Used
* *Python*: For rapid prototyping and high-level data manipulation.
* *C++*: For low-level memory management and performance-critical logic.
* *JavaScript*: For functional programming patterns and web-based logic.

### Setup & Execution
1. **Clone the repo**
```bash
git clone https://github.com/SadCloud03/daily-challenges-df-lenguages.git
cd daily-challenges-df-lenguages
```

2. **Navegate to a challenge**
```bash
cd Easy #example
```

3. **Run the file**
```bash
# Python
python3 solution.py

# C++
g++ solution.cpp -o solution && ./solution

# Node.js
node solution.js
```
